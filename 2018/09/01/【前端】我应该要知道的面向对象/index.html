不同

  • ES5中通过prototype定义的方法是可以枚举的，但是Class中内部定义的方法是不可枚举的。
    Object.keys(person.prototype); // ES5中返回[‘getHeight’] Class中返回[];
    Object.getOwnPropertyNames(person.prototype); // ES5中返回[‘getHeight’] Class中返回[‘getHeight’];
