       这篇文章用来总结一下项目做的过程中遇到的问题，不定期更新：

ios下白屏

首先，排除是否是报错，如果不存在报错，考虑JS高版本语法造成的不兼容问题。
其次，在配置好babel转译后如果仍然出现问题，建议先打开dist内的源码，查看是否还存在箭头函数。
第三，如果上述都已排除，建议查看一下 Npm 包内的第三方库类是否存在版本过高的情况。我就发生过引用过高的swiper导致白屏，需要手动去对swiper转译。高版本JQ也容易存在问题，所以建议对第三方的库类在安装的时候，强制设置版本号。

autoprefixer导致移动端多行省略号失效

查看打包后的源码，比较本地源码，发现-webkit-box-orient被打包干掉了。再细致去查这个问题，发现autoprefixer存在自动删除它认为过时的代码的问题，需要手动去关闭。
